The Web3 home page is located at http://{server}/web3/ and shows a list of the current topic maps and a list of the current schemas. From this page it is possible to create new schemas, new topics maps and edit existing schema.

Note:  Many features of the Web3 platform require that a topic map has an associated schema, it is necessary to create a schema before creating a topic map.

Web3 Home Page

Clicking on a schema name will take you to the Schema Home Page. Clicking on a topic map name will take you to the Topic Map Home Page.

If you have sufficient privileges, then the properties of a schema or topic map can be edited by clicking on the edit properties icon and the item can be deleted by clicking on the delete icon .

If you have write access to the topic map store, the New Topic Map button allows you to create a new topic map, and if you have write access to the schema store, the New Schema button allows you to create a new schema.
