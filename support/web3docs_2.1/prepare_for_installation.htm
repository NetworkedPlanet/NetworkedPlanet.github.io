Prepare for installation

 Top  Previous  Next

Before installing the software please ensure that:

 

1.All pre-requisite software and features have been installed and configured and if a machine restart was required, that restart is performed before running the Web3 Platform installer.
2.The user account you use to run the installation package has Administrator access to the machine you are installing on.
3.The user account you use to run the installation package has at least Security Administrator access to the SQL Server instance that hosts the Web3 database - this is required to add the service account to the access controls for the database.
4.You have identified or, if necessary, created a user account to use to run the Web3 services - you will be prompted for this account name and password during the installation process. This user account will be used as the identity for the Web3 services and the Web3 application pool.
