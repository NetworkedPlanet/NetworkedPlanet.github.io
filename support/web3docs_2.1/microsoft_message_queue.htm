The following MSMQ features must be enabled on all machines that host Web3 services.

•Microsoft Message Queue (MSMQ) Server Core. This feature can be found under Microsoft Message Queue (MSMQ) Server in Windows 7 and Windows Server 2008 R2. On Windows Server 2008, this feature is named Message Queuing Server and can be found under Message Queuing Services.
