The Topic Types tab in the left-hand panel allows you to browse the topic map by topic type and to create new topics of a particular type.

To browse the topic map by topic type, expand the Topic Types tab in the left-hand panel of the topic map editor. This panel shows a tree that reflects the superclass-subclass type hierarchy in the topic map schema (1). To display topics of a particular type just click on the type in the tree and the middle panel will then show the topic instances list (2). The topic instances list can be filtered by label by typing into the Filter text box.  Each topic in the topic instances list has a number of action buttons associated with it. The button displays the topic details in a dialog box in the same window. If you hold down the Alt key while clicking, then the topic details are displayed in a new browser window instead. The button loads the topic into a Topic Editor in the right-hand panel of the topic map editor window. The button will delete the topic from the topic map (you will be prompted to confirm this action before it is performed). Clicking on the Create New button brings up a dialog which prompts you for a label for the new topic to be created.

NOTE: The topic instances list will only show topics which have a label in the language selected in the language selector at the top of the page (3). If you see no topics but expected to, check you have got the editor in the correct language!
