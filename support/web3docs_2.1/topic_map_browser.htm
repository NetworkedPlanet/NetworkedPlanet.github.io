The topic map service provides not only a way to fetch topic map data in XML or JSON it also delivers HTML so that users can search and navigate the topic map using a web browser.

 

Once the Web3 Platform is installed navigating to http://{server}/web3/topicmaps/ will bring up a list of all topic maps being managed by that Web3 instance as shown in the screenshot below.

Note: the trailing '/' on the URI is important - as navigating to http://{server}/web3/topicmaps (without the trailing slash) will display the WCF service information for the Web3 topic map service.

 

Clicking on the name of a topic map in this list will take you to the Topic Map Home Page for that topic map.
