eventHandler

 Top  Previous  Next

The eventHandler element allows the configuration of external custom event handlers that are invoked when updates are made to the topic map store. The content of this element must be the fully qualified class name of the event handler followed by a comma and the fully-specified assembly reference for the assembly that contains the class. More information about developing custom event handlers can be found in the Developer's Guide.

 

The content of this element is TEXT.

 

This element occurs as a child of the topicMapStore element.
