The Complex Associations List Page shows all associations where the current topic (whose name is displayed in the main header) plays a specific type of role in a specific type of association.

 

Complex Associations List Page

 

The main header shows the name of the current topic (1) as a link which will take you back to the Topic Page for the topic. Beneath that is shown the name of the association type and the type of role played by the current topic (2) - these are also links to the Topic Page for those typing topics.

Beneath this is shown a list of associations (3). Each association is displayed in its own box (one association is shown in this screenshot). The box lists all of the topics that participate in the association and the type of role that they play. Once again the topics are each rendered as links to the Topic Page for that topic.
