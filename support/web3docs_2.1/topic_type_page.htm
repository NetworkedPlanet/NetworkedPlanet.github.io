The Topic Type Page displays information about all topics of a particular type.

Topic Type Page

The Topic Type Page displays two different lists of topics of a specific type. At the top of the page the name and subject identifier of the type is displayed (1). To the right is displayed a list of the 50 most recently edited topics of this type (2). To the left is displayed an alphabetically ordered list of the topics of this type (3). This list can be filtered by a text string typed into the search box (4). New topics of this type can be created by typing a name for the topic into the text box below this list (5) and clicking on the . On creation the new topic will automatically appear in the Recent Instances list. The add new topic button is only displayed if the user has write permission on the topic map.

Clicking on the name of a topic in either list will take you to the Topic Page for that topic.

 

NOTE: The entry panel for adding a new topic will not be displayed if you do not have write privileges on the topic map or if the topic type is marked as an Abstract Type in the topic map schema.
