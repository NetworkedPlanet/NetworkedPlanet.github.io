SQL Server with Full Text Indexing

 Top  Previous  Next

The Web3 Platform depends on SQL Server for back-end data persistence. SQL Server must be installed on the machine that hosts the Web3 Platform databases (which may be the machine on which the installer is run or on a remote machine). The supported database back-ends are:

 

•SQL Server 2008 (or later) with the full-text indexing feature enabled.
•SQL Server Express 2008 (or later) "Database with Advanced Services Edition" - this is the edition of SQL Server Express 2008 that supports full-text indexing.
