The Topic Types List displays the schema types that are allowed for new topics created in the topic map. This list is shown highlighted in red in the screenshot below.

 

 

To create new topics of a specific type or to see the topics of that type, click on the topic type name to go to the Topic Type Page.

 

If your topic map contains a large number of topic types, the list can be paged through using the grey arrow icons located at the top and bottom of the list.
