Label Editor

 Top  Previous  Next

Label editors allow you to enter the labels for a topic in multiple languages. Each language is represented with a separate label editor, with the currently selected editor language being shown as the main label editor (larger than the others). The language for each label editor is shown as a language code displayed in brackets after the text entry part of the field.

Label editors, one with text entered but not yet saved.

To begin entering text in a label editor you can either click on the icon or click directly on the text entry part of the field and then start typing.

To finish entering text and save, either click on the icon or press the Return key.

To cancel entering text and revert the field to its previous value, either click on the icon or press the Escape key.
