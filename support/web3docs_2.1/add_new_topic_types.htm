Adding New Topic Types

 Top  Previous  Next

To add a new topic type simply drag the topic type tool from the palette on the left onto the main canvas. This brings up the new type dialog where you must enter a label and subject identifier for the type. If your schema has a Base Identifier, then a subject identifier will be generated for you after you enter the label for the type this generated identifier can then be edited if you wish or deleted and replaced with your own identifier.

 

New Schema Type Dialog

 

Multiple identifiers can be assigned to a type by clicking on the ... button to bring up the Identifiers Editor.

Clicking on the Extension Properties tab will switch to the Extension Properties Editor which allows you to create additional meta data properties for the schema type.

When you are happy with the label and identifiers assigned to the topic type, click the OK button. The topic type then appears on the canvas.
