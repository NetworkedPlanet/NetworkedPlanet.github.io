ConstraintPropertyRepr

 Top  Previous  Next

Description

Represents a single extension property name/value pair attached to a constraint.

 

XML Representation

<ConstraintPropertyRepr>

  <Name>%constraint property name%</Name>

  <Value>%constraint property value%</Value>

</ConstraintPropertyRepr>

 

JSON Representation

{ 

 "Name":"%constraint property name%", 

 "Value":"%constraint property value%"

}
