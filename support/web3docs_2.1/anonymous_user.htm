Anonymous User

 Top  Previous  Next

The Anonymous user is a default user account that is added to the Web3 Platform security database on installation. This user has the user name "Anonymous" and by default is granted full privileges on the Web3 database. Whenever a user attempts to access the Web3 system, the privileges they have are equivalent to all privileges assigned to them and any roles they are in, plus all privileges assigned to the Anonymous user. The Anonymous user is also used to determine privileges when an unauthenticated request is made to the Web3 system.

 

It is therefore imperative to lock down Web3 before moving a server out of the development environment.
