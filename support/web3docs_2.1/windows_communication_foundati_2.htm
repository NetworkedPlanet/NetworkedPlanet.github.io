The following WCF features must be enabled on all machines that host Web3 services.

•Windows Communication Foundation HTTP Activation. This feature can be found under Microsoft .NET Framework 3.5.1  in Windows 7 and Windows Server 2008 R2. On Windows Server 2008, this feature is found as .NET Framework 3.0 Features > WCF Activation > HTTP Activation.
