Web3 Hive Directory

 Top  Previous  Next

All configuration and logging files for a Web3 Platform installation are contained in the Web3 Hive Directory. The standard location of this directory (created automatically by the Web3 Platform installer) is %WINDIR%\Program Files\Common Files\Web3. Under this directory are two sub-directories. The directory named LOG contains logging output from the Web3 Platform. The directory named CONFIG contains the configuration files for the services. The table below lists the standard Web3 Platform logging and configuration files.

 

Note: Other optional plug-ins and components may place additional configuration or log files under the Web3 Hive directory - refer to the documentation for those components for more details about the contents of those files.
