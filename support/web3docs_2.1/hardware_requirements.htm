Hardware Requirements

 Top  Previous  Next

The hardware requirements for Web3 depend on the role(s) played by the machine in the deployment. We identify three key roles:

•Database Server - hosts the Web3 databases
•Web3 Front-End Server - hosts some or all of the Web3 services, connecting to a remote database server
•Web3 Combined Server - hosts the Web3 databases and all Web3 services

 

A Developers Edition installation that includes its own SQL Server Express database should meet at least the minimum requirements for the Database Server role, but when combined with running Visual Studio and other developer tools it should at least meet the Recommended Requirements for the Web3 Combined Server role.
