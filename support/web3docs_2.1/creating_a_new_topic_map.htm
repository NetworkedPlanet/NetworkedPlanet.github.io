Creating A New Topic Map

 Top  Previous  Next

To create a new topic map click on 'Create New Topic Map'. This will bring up the following dialog.

Create Topic Map Dialog

 

Provide a topic map label in one or more of the supported languages by selecting the language from the drop-down list (1) and entering the label in the label text box (2).

You can optionally provide a Web Label for the new topic map (3). A Web Label is a short-cut to the topic map, allowing you to access it by its web label rather than its unique identifier - for example if you set the Web Label to "employment" then the topic map would be accessible at http://{server location}/topicmaps/employment as well as at http://{server location}/topicmaps/{topic map unique id}.

You must also select a schema for the topic map from the drop-down list (4).

Finally, click OK to confirm and create the topic map or Cancel to exit the dialog without creating a topic map.

When you click OK, the topic map is created on the server in two stages, first the topic map is created and then it is synchronized with the schema. The process of synchronization creates a copy of the topic map schema inside the topic map itself ensuring that if further changes are made to the schema, they will not affect the topic map until you choose to repeat the synchronization process.
