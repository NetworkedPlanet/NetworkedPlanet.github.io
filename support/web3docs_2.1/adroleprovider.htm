adRoleProvider

This element wraps the configuration information required to use the Active Directory Role Provider. This section is only used if the Active Directory Role Provider is configured for use in the roleProvider element in the securityManager section of the configuration file. This element contains the following child elements:

 

•rootPath: OPTIONAL, NON-REPEATABLE
•userName: OPTIONAL, NON-REPEATABLE
•password: OPTIONAL, NON-REPEATABLE

 

This element is a child of the server element.
