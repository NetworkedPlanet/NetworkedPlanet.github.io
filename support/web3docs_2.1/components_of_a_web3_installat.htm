The diagram below shows the different components of a Web3 installation and their logical groupings.

Web3 Components

 

•SQL Server components are the Web3 databases.
•Web3 Server Components are the applications, configuration files and other files that provide all of the Web3 services. These can be broadly grouped into the web application (the Portal, Topic Map Service and Schema Service) and the Windows Services (the Jobs Service and Cache Service).
•Web3 SDK components are libraries used to build client applications that interact with the Web3 server.
•Web3 Admin components are client-side tools that are provided by a separate Web3 Admin Tools installer (they are not part of the Web3 installer).

 

The rest of this section will describe some of the different ways in which these components can be installed to service different requirements in terms of scaling, distribution and licensing.
