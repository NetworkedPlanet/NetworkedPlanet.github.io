eventThreshold

 Top  Previous  Next

This element specifies the level of logging for messages written to the system Event Log. Event log messages can be found in the Application Log section of the Event Log. The content of this element is one of the following logging levels (listed from most verbose to least):

 

•ALL
•DEBUG
•INFO
•WARN
•ERROR
•FATAL
•OFF

 

The default value (used if this element is omitted) is WARN.

 

This element is a child of the logging element.
