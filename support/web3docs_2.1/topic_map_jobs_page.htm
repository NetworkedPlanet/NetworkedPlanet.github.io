The Topic Map Jobs Page lists all the jobs that have executed, or been scheduled for execution along with their progress percentage (if available) and current processing status.

Topic Maps Jobs Page.

The 'Refresh' button will check the status of all listed jobs and reflect any changes in the display.

If the user has adequate permissions on the topic map then the 'Synchronize Schema' button will be visible. Clicking this button will re-sync the topic map with it's specified schema.

Clicking on a specific job will take the user to the Job Details page.
