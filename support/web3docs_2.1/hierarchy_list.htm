The hierarchy list displays all of the hierarchy definitions in a particular topic map. This list can be found on the topic map home page. It is shown outlined in red in the screenshot below.

 

 

To view the content of a hierarchy, click on the hierarchy name to go to the Hierarchy Navigator.

If the topic map you are viewing has a large number of hierarchies defined in it the list of hierarchies can be paged through using the grey arrow icons at the top and bottom of the list.
