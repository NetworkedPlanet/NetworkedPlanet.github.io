Removing Types and Constraints

 Top  Previous  Next

Types can be removed from the schema in one of two ways, depending on what kind of Type it is.

•Topic Types, Association Types, Arc Types and Role Types can be deleted from the schema by selecting them in the main Schema Editor panel and pressing the Delete key.
•Name Types and Occurrence Types do not appear on the Schema Editor panel. To delete these types you must locate the type in the Navigator panel and double-click on it to bring up the properties dialog; this dialog includes a Delete button which will remove the type from the schema entirely. Any topic types which reference a deleted Name or Occurrence Type through a constraint will have that constraint removed.
