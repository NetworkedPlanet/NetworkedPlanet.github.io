The Topic Map Search Page is used to search through the topics in a topic map by their names or label values.

Topic Map Search Page

A search tern can be entered in the text box at the top of the page. The search supports full word matches by entering the text to match, or wildcard matches using a *. So "eu" will not find the topic "Europe", but "eu*" will.

Searches can be restricted to only specific types of topic by selecting the topic type from the drop-down list.

The results are displayed in the main part of the page. Use the drop-down list at the bottom of the results to change the number of results displayed per page and use the grey arrow icons at the top and bottom of the results list to navigate through pages of results. Clicking on a topic displayed in the results set will take you to the Topic Page for that topic.
