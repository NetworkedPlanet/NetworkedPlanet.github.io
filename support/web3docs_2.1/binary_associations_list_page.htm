The Binary Association List Page shows all topics that play a particular type of role in a particular type of association with the current topic. In the sample below it shows all topics that play the role of "Friend" in a "Friendship" association with the current topic.

Binary Associations List Page

The main title of the page shows the type of the association that relates the listed topics to the current topic (1). Beneath that is shown the name of the current topic followed by the name of the type of role that the listed topics play in the relationship with the current topic (2). The list itself is a simple paged list of all the related topics (3). As with other paged lists, the number of topics per page can be controlled with the drop-down list at the bottom of the list and the pages are navigated using the grey arrow icons at the top and bottom of the list.
