fileThreshold

 Top  Previous  Next

This element specifies the level of logging for messages written to the log file in the Web3 Hive Directory. The content of this element is one of the following logging levels (listed from most verbose to least):

 

•ALL
•DEBUG
•INFO
•WARN
•ERROR
•FATAL
•OFF

 

The default value (used if this element is omitted) is DEBUG.

 

This element is a child of the logging element.
