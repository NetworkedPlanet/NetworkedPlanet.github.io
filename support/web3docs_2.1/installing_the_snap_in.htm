Installing The Cmdlets

 Top  Previous  Next

The Powershell cmdlets are provided by a PowerShell snap-in called "Web3Admin". This snap-in is provided in a separate installer from the main Web3 installer to allow the snap-in to be installed on any machine. You are allowed to install the snap-in on as many machines as you need to.

 

To Install The Snap-In

From the Web3 installation package you downloaded locate the Web3Admin directory. This contains a single MSI installer file named Web3AdminSetup.msi. Launch this MSI installer by double-clicking on it and follow the prompts. Please note that the 32-bit download package contains only the 32-bit Web3Admin MSI and the 64-bit download package contains only the 64-bit Web3Admin MSI. Please ensure that you use the correct installer for your machine so that the snap-in is correctly registered.
