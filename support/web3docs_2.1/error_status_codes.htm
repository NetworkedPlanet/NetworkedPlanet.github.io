Error Status Codes

 Top  Previous  Next

The Web3 server uses standard HTTP status codes where possible:

•404 Not Found - returned when attempting to access a non-existent resource.
•401 Not Authorized - returned when the user does not have the necessary privileges for the requested operation.
•406 Not Acceptable - returned when the requested content type is not supported by the resource
•400 Bad Request - returned when the content of the request could not be parsed as the expected type or if there is a syntax error in a SPARQL query string.
