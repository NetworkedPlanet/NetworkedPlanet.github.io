The Web3 Platform server runs on IIS. IIS 7.0 or later is required. The following features must be enabled in the IIS installation:

•ASP.NET (under Internet Information Services > World Wide Web Services > Application Development Features)
•Static Content (under Internet Information Services > World Wide Web Services > Common HTTP Features)
•HTTP Errors (under Internet Information Services > World Wide Web Services > Common HTTP Features)
•HTTP Redirection (under Internet Information Services > World Wide Web Services > Common HTTP Features)
•Default Document (under Internet Information Services > World Wide Web Services > Common HTTP Features)
•IIS Metabase and IIS6 Configuration Compatibility (under Internet Information Services > Web Management Tools > IIS 6 Management Compatibility)
•Windows Authentication (under Internet Information Servicese > World Wide Web Services > Security)

 

Note: On Windows Server 2008, IIS is installed by activating the Web Server (IIS) role.
