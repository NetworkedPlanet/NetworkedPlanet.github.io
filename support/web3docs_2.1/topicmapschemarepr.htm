Description

Representation of the schema attached to a topic map.

 

XML Representation

<TopicMapSchema 

 Address="%schema resource uri%" 

 TopicMapAddress="%topic map resource uri%" 

 LastUpdated="2010-07-24T13:23:46.3785435Z">

  <SourceSchemas>

    <SchemaSummary ...> See SchemaSummaryRepr </SchemaSummary>*

  </SourceSchemas>

  <Types>

    <Type ...> See MergedTypeRepr </Type>*

  </Types>

  <Constraints>

    <Constraint ...> See MergedConstraintRepr </Constraint>*

  </Constraints>

  <Properties>

    <Property ...> see TypePropertyRepr </Property>*

  </Properties>

</TopicMapSchema>

 

JSON Representation

 

{ 

"Address":"%schema resource uri%", 

"TopicMapAddress":"%topic map resource uri%", 

"LastUpdated":%date/time%, 

"SourceSchemas": array of SchemaSummaryRepr,

"Types": array of MergedTypeRepr,

"Constraints": array of MergedConstraintRepr,

"Properties" : array of TypePropertyRepr

}
