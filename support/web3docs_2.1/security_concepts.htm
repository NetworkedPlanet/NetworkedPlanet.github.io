This section covers the basic concepts require to understand how the Web3 Platform implements access controls and user authentication. In this section:

 

•Stores and Partitions- describes the different data containers that access controls apply to.
•Authentication - describes how the Web3 Platform determines the identity of a user from their request.
•Authorization - describes how the Web3 Platform determines what permissions a user has on a data container.
