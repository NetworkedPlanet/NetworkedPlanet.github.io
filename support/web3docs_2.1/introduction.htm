Thank you for installing the Web3 Platform from NetworkedPlanet.

 

This document provides useful information for administrators, developers and users and so is split into multiple sections.

 

For Administrators:

 

•The Installation section covers all aspects of planning for and deploying the Web3 Platform.
•The section on Adminstration covers more detailed tasks such as configuration and security management as well as providing a reference to the PowerShell cmdlets provided for administrators.

 

For Developers:

 

•The Developers Guide provides an overview of the use of the different APIs provided by the Web3 Platform and a reference guide to our basic REST interface.
•A full .NET API reference can also be found as a separate HTML Help file, or online at http://support.networkedplanet.com/docs/web3/api.

 

For all users:

 

•The User Guides cover the use of the standard Web3 front-end tools for schema and topic map browsing and editing.
