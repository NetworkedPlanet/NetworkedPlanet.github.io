The Arcs List Page shows a list of all topics that are connected to a particular starting topic using a particular type of arc.

Arcs List Page

The name of the arc type is displayed in the page title area (1) beneath this is shown the name of the starting topic (2) as a link back to the topic page for that topic. Underneath this is a list of the topics that are connected to the starting topic by arcs of the specified type (3). If the list is very long paging controls are displayed to see more topics. Each topic is displayed as a link to the topic page for that topic.
