NP_TM_TopicType

 Top  Previous  Next

The value of this configuration attribute is the name of the page property that must be defined on topic-aware page types. The default value is nptopictype. To make a page type topic-aware you must create a page property with the value of this attribute.

For a sample definition of the property referenced by this setting, please refer to the section Mapping Page Types to Topic Types.

Do not confuse this configuration attribute with NP_TM_TopicPsi. This attribute names the page property that holds the identifier of the type of topic for a page - it is usually defined on a page type with a default value and is not usually modified by editors, so all pages of the same type will have the same value for this property. NP_TM_TopicPsi names the page property that holds the identifier of the specific topic for a particular page, this property will be different for each page.
