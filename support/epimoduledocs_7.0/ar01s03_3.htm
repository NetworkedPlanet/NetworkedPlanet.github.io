This section describes the behaviour and configuration of the web parts that are supplied with this module. The following section provides a simple walk-through of creating a faceted search page template.

The value of the TMIS is derived from the ASP.NET 2.0 Web Parts that are supplied with the module. All the web parts are to function outside of a web part environment as regular controls (i.e. "standalone" mode in Web Part parlance).
