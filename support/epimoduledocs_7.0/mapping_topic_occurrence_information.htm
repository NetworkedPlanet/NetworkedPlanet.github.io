Topic Occurrence Information

Each page topic contains two occurrences:

1.The ID of the page in EPiServer.
2.A URL of the page within EPiServer.

The occurrence data is not critical to the operation of the TMCore EPiServer Module, however it is very useful for developers who wish to reference or link to EPiServer content without having to make a direct connection to EPiServer to query such information.
