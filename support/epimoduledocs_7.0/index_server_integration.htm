Topic Map Index Server Integration

 Top  Previous  Next

The Topic Map Index Server (TMIS) by Networked Planet provides a standalone index service for topic maps held in TMCore. The TMIS provides high speed access and query facilities for topic maps through the Windows Communication Foundation framework introduced in .NET 3.0.

The TMCore EPiServer Module has an extension assembly which allows site developers to take advantage of the TMIS facilities via new ASP.NET 2.0 Web Parts (note that there is no public API for the TMIS in this version of the module):

1.Faceted Navigation
2.Faceted Search
3.Topic Clouds
