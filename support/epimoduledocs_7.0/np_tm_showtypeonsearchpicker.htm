NP_TM_ShowTypeOnSearchPicker

 Top  Previous  Next

This property determines whether the page type and/or topic type names should be shown in the table of potential topics to create associations to on the EPiServer Edit Mode Topic Map tab. Allowed values are None, TopicTypeOnly, PageTypeOnly, or Both (case is not important).

This property only applies to the legacy editing interface. The new CMS 7 editing interface always displays the topic type name.
