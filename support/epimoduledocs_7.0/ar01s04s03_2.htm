Run the EPiServer Module SQL script

 Top  Previous  Next

The TMCore EPiServer Module requires new stored procedures to be added to the database. Using SQL Query Analyzer or some other tool execute the script called EPiServerModule.sql that can be found in <InstallFolder>\sql, against your TMCore database.

The script provided does not specify the database that it runs against. When running the script always be sure that you have specified the correct TMCore database before executing the script.
