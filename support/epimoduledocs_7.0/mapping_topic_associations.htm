Topic Associations

 Top  Previous  Next

Each association created in the Edit mode Topic Map component (or Topic Map tab in the legacy editing UI) is modeled using a standard topic map association. Therefore associations typically connect one page topic to another page topic. In addition each association including a scoping topic, which is used to mark the association as deleted or unpublished.

EPiServer has a concept of a published page (which is viewable to everyone) and the unpublished (working copy) page. Editors work on the working copy of a page before publishing it. The TMCore EPiServer Module uses the Topic Maps concept of scope to support this model. The scope is used to track changes being made to the associations to/from a page prior to the page being published.
