This chapter describes how EPiServer editors use the TMCore EPiServer Module as part of the content authoring process. From version 7 of the TMCore EPiServer Module, two different editor interfaces are supported. The new editing interface is integrated with the CMS 7 editing interface and is available as an optional component that can be displayed in the tabs at the side of the page. The legacy editing interface is only available through the legacy editing mode in CMS 7 and can be found on the 'Topic Map' tab in the page edit screen.
