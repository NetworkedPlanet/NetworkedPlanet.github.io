Page Link Occurrence

 Top  Previous  Next

An occurrence is created on each page topic containing a resource link which is the relative URL for the page, as read from the PageData.LinkURL property). The occurrence is typed using a topic with the subject identifier http://www.networkedplanet.com/2005/01/episerver/livepage (or the value of the NP_TM_PageLinkOccurrenceTypePsi configuration parameter).

Example:

Resource

/EPi/en/Topics/Intermediate_Language_Engineering/

Type

http://www.networkedplanet.com/2005/01/episerver/livepage
