Topic Name Information

 Top  Previous  Next

EPiServer has powerful internationalization and globalization support as standard. Each page may have several variants in different languages. The property we are concerned with here is the name of the page.

For each language branch present for a page a single name is created on the page topic scoped by a topic which represents the language the page is in. For example: if a page is created with variants in English, Norwegian (Norsk), and Finnish (Suomi) then the page topic will have 4 scoped names. Each topic will contain a single scope created from the language branch name:

•Topic Maps (en)
•Emnekart (nb-NO)
•Aihekartta (fi-FI)

In addition to this a default (unscoped) name will be created which is taken from the default language.
