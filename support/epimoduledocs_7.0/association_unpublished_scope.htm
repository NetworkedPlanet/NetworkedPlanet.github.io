Association Unpublished Scope

 Top  Previous  Next

When an editor creates an association from within the EdIt Mode Topic Map tab the association is created in the map, but with an extra scoping topic added to the assocation. This topic has the subject identifier http://www.networkedplanet.com/2005/01/episerver/association-unpublished. When a user views any existing published page the presence of this scoping topic prevents the association from being displayed. Once the page at either end of the association is published then this scope is removed and the association becomes visible to all.

Note that when using the editing component, associations marked as deleted or unpublished will be shown to enable the editor to see all the changes to be made before the page is published. For more information on this please refer to the Editors' Guide.
