These are the components to the EPiServer Module:

1.The EPiServer Plug-Ins. There are two plug-ins for the EPiServer Module. One creates adds the “Topic Map” tab in the legacy editing interface and adds the “TMCore Admin” tabs in the EPiServer Admin interfaces. The other adds a "Topic Map" component to the new EPiServer CMS 7 editing interface.
2.EPiServer Module control helpers. This is a collection of function that allow easy and rapid development of EPiServer solutions that exploit the EPiServer Module.
