This should be set to the path, on the local web server to where the aspnet_client files are installed. If you follow the default installation you will need to change this to point to the aspnet_client directory inside the EPiServer web application, for example /EPI/aspnet_client/NetworkedPlanet.EPiServerModule/7.0/.

You can confirm that you have the correct path by entering it in to the web browser and adding the name of a file inside that directory, for example:

http://localhost/EPI/aspnet_client/NetworkedPlanet.EPiServerModule/7.0/topiceditor/css/common.css

Default value: /aspnet_client/NetworkedPlanet.EPiServerModule/7.0/

You cannot use a tilde (~) character within this URL.
