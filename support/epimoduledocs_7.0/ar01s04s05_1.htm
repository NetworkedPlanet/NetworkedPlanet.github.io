Select a topic map for use

 Top  Previous  Next

The TMCore EPiServer Module requires a topic map. By default it expects this topic map to be called “EPi”, however this can be overridden in the EPiServer web.config file using the NP_TM_TopicMapName appSetting property. If you do not have a topic map created, then you may do so with the TMCore TMEditor web application. For information on the Topic Map Editor and defining a schema see the relevant documentation in the TMCore release.
