Using the Page ID Search Picker

 Top  Previous  Next

The Page ID Search Picker allows you to make a relationship to another page in EPiServer by entering the page ID of the related page. The main components of the Page ID Search Picker are identified in the screenshot below:

Page ID Search Picker

1.Page ID Text Box
In this text box you can enter the numeric page ID of the page you want to find.
2.Language Filter Drop Down
With this drop-down list you can select one of the site languages to filter by. A result will only be returned if the page with the specified page ID has a variant in the language specified in the language filter. The default (empty) value will return a result if the page exists, regardless of which languages it is available in.
3.Search Results List
The search results list will show the matching page or the message "No matches found" if no matching page could be found. Only pages that have a matching topic in TMCore are displayed, if the page you specified exists but is not represented by a topic in TMCore, then no match will be displayed. If the type of the topic is correct for the currently selected related topics list, then the icon will be shown in the Add column - you can click on this to add the topic to the related topics list. If the type of the topic is not correct for the currently selected related topics list, then the message "Invalid Type" will be shown in the Add column instead.
