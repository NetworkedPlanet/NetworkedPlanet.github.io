Basic Challenge/Response issued when accessing the Topic Map tab

 Top  Previous  Next

Symptom

When accessing the Topic Map tab the editor is challenged with a pop-up asking them to log in.

Fix

The most likely cause is a lack of file permissions. You must ensure that the application pool user (i.e. the identity with which IIS is executing EPiServer) has read access to the aspnet_client directory and all of its subdirectories.
