Create Occurrence Types

 Top  Previous  Next

You will need to create one topic for each occurrence type that you identified during the analysis phase, and modify the schema of content or subject type topics to indicate where each occurrence type is used.

Create the occurrence types as you would for the other topic types. Again you can use any name for the occurrence type that you like - this is the name that will be displayed to the end-user. You must assign the type Occurrence Type to each topic as you create it from the topic types list on the right-hand side of the Topic Editor page.

Having created the topics for all of the occurrence types, return to the Topic Selector page and choose Topic Type from the left-hand list box and choose one of the types that has one or more occurrences assigned to it in your ontology from the right-hand list box and the click Edit. On the Topic Editor Page, you will see that there is tab labelled Occurrences. On this page either Value or Resource occurrences can be added to the topic. Select those occurrence types which are allowed for this topic type.
