Page ID Occurrence

 Top  Previous  Next

An occurrence is created on each page topic containing resource data which is the integer ID of the page as taken directly from EPiServer. The occurrence is typed using a topic with the subject identifier http://www.networkedplanet.com/2005/01/episerver/pageid (or the value of the NP_TM_PageIdOccurrenceTypePsi configuration parameter).

Example:

Data

67

Type

http://www.networkedplanet.com/2005/01/episerver/pageid
