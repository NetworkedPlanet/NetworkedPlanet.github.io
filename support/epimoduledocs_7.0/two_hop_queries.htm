Two Hop Queries

 Top  Previous  Next

A Two Hop Query is used to find all pages that are indirectly connected to a starting page via some other page. So the query follows one hop from the starting page to some intermediate page and then a second hop from the intermediate page to the final page. For each of these hops you specify the subject identifier of the association type and role type you wish to navigate. The hops can use different types and each hop can produce multiple results.
