Redundant Topic Removal

 Top  Previous  Next

If the topic map service misses any page update events - because it has been disabled for example, the topic map may contain redundant topics that point to pages in EPiServer that are no longer valid. If this is suspsected in the topic map, then the Redundant Topic Removal job can be run. This job will find all topics in the topic map that contain references to pages in EPiServer and checks whether they still are valid and point to existing pages in the site. Any topics that are found to be for invalid pages are removed from the topic map, including any references to assocations that the topic was part of. This job can be resource intensive so it is recommended it only be run when redundant topics are suspected to have been created in the topic map and not as a recurring scheduled job.
