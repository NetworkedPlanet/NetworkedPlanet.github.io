Editing Related Topics

To add a new topic to a related topics list:

1.Click on the related topics list you wish to add. This list should now get a green background as shown below:
2.Use one of the pickers to find the related topic you want to add to this list and click on the icon.
3.If the topic you selected was not already in the list, it will now appear in the related topics list shown in italics to indicate that it is a new addition that will only be visible on the site when the topic changes are published. For more information about publishing topic changes please refer to the section Saving and Publishing Changes. In the screenshot below the variety "Balmoral" has been added to the list.
4.If you mistakenly added the wrong topic you can click on the icon to remove to the topic from the related topics list.

 

To remove a topic from a related topics list:

1.Click on the related topics list you wish to edit. The list should now get a green background.
2.Click on the icon to delete the topic.
3.If the topic is a published related topic (shown in a normal font), it will be recorded as marked for deletion when the topic changes are published and displayed with strike-through. If the topic is a new related topic (shown in italics) that has not yet been published it will simply be removed from the list. For more information about publishing topic changes please refer to the section Saving and Publishing Changes. In the screenshot below, the variety "Kelvedon Wonder" has been marked for deletion.
4.If you mistakenly mark a published topic as deleted you can undelete it by clicking on the icon again. If you mistakenly delete an unpublished topic you will have to use the topic pickers to find and add the topic again.
