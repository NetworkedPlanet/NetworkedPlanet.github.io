Topic List Picker

 Top  Previous  Next

The topic list picker allows the editor to see a list of all the potentially associated topics and refine by a text search. Use % and _ for wildcards (any number of characters or single character). To specify the language of topics to display, a Language dropdown allows the editor to view in a language of their preference. To view all topics with no scope on the name, an Unscoped language option will allow the editor to see all topics with unscoped names.

Topic List Picker

Before selecting a topic an editor may wish to check or differentiate between several topics in the list. By clicking anywhere on the topic name a new display appears at the bottom of the screen that displays details about the topic.
