NP_TM_DefaultPickerList

 Top  Previous  Next

This allows administrators to define a default set of pickers that will be shown to the editor when classifying pages in EPiServer. The default value is: SelectTopicFromInstanceList,HierarchyPicker,IDPicker. See the Administrators' Guide for more information about this configuration option.

This property applies only to the legacy editing interface. The new CMS 7 interface always displays the three standard pickers.
