NP_TM_ShowTypeOnExistingAssociations

 Top  Previous  Next

The value of this property determines whether the page type and/or topic type names should be shown when listing the current associations on the EPiServer Edit Mode Topic Map tab. Allowed values are None, TopicTypeOnly, PageTypeOnly, or Both (case is not important).

This property only applies to the legacy editing interface. The new CMS 7 editing interface displays only the topic name without any type information.
