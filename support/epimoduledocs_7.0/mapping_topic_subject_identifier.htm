Topic Subject Identifier (URN GUID)

 Top  Previous  Next

The page topic subject identifier is critical to the operation of the EPiServer Module as it is the unique identifier within the topic map that the EPiServer module uses to identify the page topic. It is composed of a fixed prefix urn:guid:, followed by a globally unique identifier (or GUID).

Example: urn:guid:a102c562-6dc1-44f5-98fd-bf95f5da458c

The urn:guid: prefix is the default prefix used by the TMCore EPiServer Module. We recommend that you always use this default, however if you wish to use some other prefix you can do so by configuring the NP_TM_URNGuidPrefix application setting. For information about this and all other application settings please refer to the AppSettings section of the Installation Guide.

This property once created on a page should not be modified by any action other than through the EPiServer module itself. If you create code that modifies this property you run the risk of disconnecting a page in EPiServer from its topic, which will lead to unexpected duplications and query failures.
