Querying Data
=======================================================

A query editor is embedded on the Data Platform to help users format queries to run against the datasets on the Data Platform.

.. figure:: /images/query.png
   :scale: 90 %
   :alt: Querying datasets
   
Type a SPARQL query into the editor and then press the "Submit" button to run the query against the selected dataset (or all datasets in the Data Platform).
